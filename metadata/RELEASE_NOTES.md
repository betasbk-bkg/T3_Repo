# Release notes: v1.0.1

Clean reproducibility package for the manuscript “Projected anti-aggregate stabilization and mistiming-cost boundaries in latency-affected quantized collective control”.

Included:

- Deterministic simulation engine for latency-affected quantized collective control.
- Confirmatory simulation script.
- Closure diagnostic script for the 96-cell grid.
- Second-pass diagnostic regression/classification analysis.
- Bootstrap boundary-check script and processed bootstrap outputs.
- Final manuscript figures Figure_1.png–Figure_7.png, with Figure 1 updated to remove cropping and improve readability.
- README, manifest, Zenodo description, citation template, and metadata template.

Important version-specific manuscript numbers:

- 96 total closure cells.
- 78 benefit cells.
- 18 harm/non-benefit cells.
- 12/12 benefit cells at design latency d=26 frames.
- 15 harm-cost-dominance cells.
- 1 harm cell with local-relief reversal.

Notes:

- This release supports the manuscript framing in which the mechanism is projected local error relief under normalized vector-mean aggregation, not physical speed damping.
