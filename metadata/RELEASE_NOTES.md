# Release notes: v1.0.0-t3-cnsns

Initial CNSNS-submission reproducibility package.

Included:

- Deterministic simulation engine for latency-affected quantized collective control.
- T3 confirmatory simulation script.
- Closure diagnostic script for the 96-cell grid.
- Second-pass diagnostic regression/classification analysis.
- Bootstrap boundary-check script and processed bootstrap outputs.
- Final CNSNS submission figures Figure_1.png–Figure_7.png.
- Manuscript, highlights, and cover-letter reference files matching the CBI REFBOOST FINALCHECK submission version.
- README, manifest, Zenodo description, citation template, and checksum manifest.

Important version-specific manuscript numbers:

- 96 total closure cells.
- 78 benefit cells.
- 18 harm/non-benefit cells.
- 12/12 benefit cells at design latency d=26 frames.
- 15 harm-cost-dominance cells.
- 1 harm cell with local-relief reversal.

Notes:

- This release supports the CNSNS submission framing in which the mechanism is projected local error relief under normalized vector-mean aggregation, not physical speed damping.
